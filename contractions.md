# Contractions

## Gramatical
| Full Phrase | Contraction |
| ------------ | ----------- |
| à   | a a (ex. eu vou **à** casa)  |
| ao   | a o (ex. eu vou **ao** cinema)  |

## Informal
| Full Phrase | Contraction |
| ------------ | ----------- |
| não é?   | né? |
| para a   | pra |
| para o   |  pro |
| meo   | meu amigo, meu _brother_  |
| pelo   | por o  |
| pela   |  por a |
